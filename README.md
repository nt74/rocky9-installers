# rocky9-installers
Rocky Linux 9 installer scripts to simplify things for the user.

## Instructions
```
mkdir -p ~/src && cd ~/src
```
```
git clone https://github.com/nt74/rocky9-installers.git
```
```
cd rocky9-installers
```
```
chmod +x install-*
```

It is now possible to run any installer script by running:

./install-*script*.sh
